# Getting Started

This guide will show you how to install FocusMode plugin with ease.

## Installation

Follow these simple steps to get started:

1. Head over to the [releases page](https://github.com/AshesOfPhoenix/focus_mode.koplugin/releases/latest)  
   and grab the latest `.zip` file.
2. Unzip the contents into your **plugins** folder.
3. Set From and To hours of the day
4. Set a protective PIN (Optional)
5. Toggle Enable
